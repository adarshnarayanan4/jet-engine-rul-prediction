# Results Folder

This folder stores graphs generated from the Colab notebook.

Included graphs:

- `sorted_all_models_rul_comparison.png`: compares actual NASA RUL against Random Forest, Gradient Boosting, Extra Trees, and XGBoost predictions, sorted by engine health.
- `all_engines_model_comparison.png`: compares actual RUL and predictions by engine ID.
- `model_error_comparison_by_engine.png`: compares absolute prediction error by engine.
